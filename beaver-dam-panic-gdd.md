# Beaver Dam Panic (비버 댐 패닉) - 해커톤 기획안

## 1. 프로젝트 개요
**"홍수 직전의 댐을 구하라! 비버와 함께하는 댐 수리 대작전"**
- **장르:** 모바일 퍼스트 웹 미니게임 (캐주얼 액션)
- **한 줄 요약:** 제한 시간 내에 수압을 견디며 댐의 구멍을 알맞은 재료로 막아 생존하는 반응형 웹 게임입니다.

## 2. 핵심 게임 플레이
- **핵심 루프:** 구멍 확인 → 알맞은 재료(나뭇잎, 나무, 돌) 선택 → 수리 → 수압 관리 → 생존 성공/실패
- **난이도 모드:**
    - **Easy Mode:** 30초 생존 (입문자용)
    - **Hard Mode:** 60초 생존 (도전형, 높은 긴장감)
- **성공/실패 조건:** 제한 시간을 버티면 성공, 수압이 100%에 도달하면 댐이 무너지며 실패

## 3. 주요 기능 (Core Features)
- **튜토리얼 (How to Play):** 게임 규칙과 조작법을 직관적인 모달로 안내
- **일시정지 (Pause):** 게임 중 긴급한 상황에서 흐름을 끊고 재개 가능
- **기록 시스템 (Records):** 모드별 최고 점수 로컬 저장 및 확인
- **결과 피드백 (Result):** 성공/실패 여부, 점수, 생존 시간 기반 별 평가(Star Rating) 제공
- **모바일 최적화 (Portrait Notice):** 모바일 세로 모드 권장 안내 및 반응형 UI 제공

## 4. 기술 스택 및 배포
- **Frontend:** React, Vite (Single-page Application)
- **Styling:** Vanilla CSS (모바일 퍼스트 반응형 설계)
- **Infrastructure:** GitHub (버전 관리), Vercel (자동 배포 및 호스팅)

## 5. AI 협업 워크플로 (AI-assisted Vibe Coding)
본 프로젝트는 기획부터 배포까지 AI와 긴밀히 협업하는 **'바이브 코딩(Vibe Coding)'** 방식으로 제작되었습니다.
- **ChatGPT Plus:** 게임 기획, 요구사항 구체화, 로직 설계 지원
- **Gemini CLI (on Termux):** 모바일 환경에서 직접적인 코드 구현 및 실시간 디버깅 수행
- **특징:** 기획자의 의도를 AI가 즉각 코드로 반영하고, 반복적인 테스트를 통해 완성도를 높이는 초고속 프로토타이핑 공정 채택

## 6. 프로젝트의 차별점
1. **설치가 필요 없는 접근성:** URL 링크만으로 모바일/PC 어디서든 즉시 플레이 가능
2. **짧고 강렬한 재미:** 30~60초 내외의 플레이 타임으로 반복 도전 유도
3. **완성도 높은 UX:** 튜토리얼, 기록 저장, 세로 모드 대응 등 미니게임임에도 필수 편의 기능 완비
4. **효율적 개발:** AI 코딩 도구를 활용해 1인 개발 환경에서 기획-개발-배포의 전 과정을 단시간에 완수
